About This Project

The Project

Astra Space Explorer is a web application built to make space exploration accessible, visually compelling, and educational. This platform connects directly to public data streams from the National Aeronautics and Space Administration (NASA), bringing historical archives and daily astronomical highlights to a modern, easy-to-use interface.

Created as a student project, the site aims to demonstrate how complex datasets—such as the tens of thousands of raw images transmitted from the surface of Mars—can be explored without unnecessary friction or overwhelming technical barriers.

Data Sources

All imagery, dates, and explanations provided on this platform are retrieved dynamically from NASA's Open APIs. We use the following endpoints:

API NASA Astronomy Picture of the Day (APOD)
Endpoint https://api.nasa.gov/planetary/apod
